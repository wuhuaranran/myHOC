# React HOC Demo

1. npm i
2. npm run start
## 说明
1. components中为组件，WithSubscription为高阶组件，BlogPost和CommentList为被包裹组件，DataSource为测试数据源。
2. pages/index.js为入口页面
3. pages/_error为404页面
